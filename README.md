Sixornot
========

IPv6 status indicator for Firefox.

For more information, see addon listing: https://addons.mozilla.org/en-GB/firefox/addon/sixornot/


To build: ```npm run build```


To run test suite:

npm install
./node_modules/mocha/bin/mocha
